# Mediverse
Mediverse is a digital health platform designed to store users medical data and ensures it's accessibility in case of emergency. User can store there medical report and can share it just by scaning QR code without revealing thier identity. They can store their critical medical data in emergency card which can be accessed by authorized hospital in case of emergency. The report uploaded on the platform are uploaded on cloud so it can be easily accessed by user also the reports are analysed by ai. This data is used in health tracking. The platform has an inbuilt ai chatbot which gives medical advice to the user.
